## 页面功能分配

### index.html - 主页面
**目标**: 吸引用户，展示核心价值，促成下载
**内容结构**:
1. **导航栏**: Logo + 导航链接 + 语言切换器
2. **英雄区域**: 
   - 动态背景效果
   - 产品标题和核心卖点
   - CTA按钮（立即安装）
3. **核心功能预览**: 
   - 一键生成提示词演示
   - 支持的AI模型展示
   - 多语言支持展示
4. **使用统计数据**: 
   - 用户数量、生成次数等
   - 动态数字滚动效果
5. **产品特色**: 
   - 轻量级、开源免费等优势
   - 图标 + 简短说明
6. **用户评价**: 
   - 轮播展示用户反馈
   - 社交媒体链接
7. **页脚**: 版权信息和联系方式

### features.html - 功能详情页
**目标**: 详细介绍产品功能，解答用户疑问
**内容结构**:
1. **导航栏**: 与主页保持一致
2. **功能概览**: 
   - 功能分类展示
   - 交互式功能演示
3. **详细功能介绍**:
   - 模型选择功能
   - 多语言生成
   - 图片尺寸过滤
   - 生成历史管理
   - 自定义平台跳转
   - 提示词模板
   - 自定义说明输入
   - 国际化界面
   - 轻量级UI
   - 画面比例预设
   - 域名过滤
   - 本地图片生成
4. **技术规格**: 
   - 支持的浏览器
   - 系统要求
   - API集成说明
5. **FAQ部分**: 常见问题解答
6. **页脚**: 与主页保持一致

### guide.html - 使用指南页
**目标**: 帮助用户快速上手使用产品
**内容结构**:
1. **导航栏**: 与主页保持一致
2. **安装指南**: 
   - Chrome/Edge安装步骤
   - 开发者模式说明
   - 截图演示
3. **配置教程**: 
   - API Key获取和设置
   - 模型选择配置
   - 个性化设置说明
4. **使用教程**: 
   - 基本使用方法
   - 高级功能介绍
   - 最佳实践建议
5. **故障排除**: 
   - 常见问题解决方案
   - 技术支持联系方式
6. **更新日志**: 版本更新说明
7. **页脚**: 与主页保持一致

## JavaScript功能模块

### main.js 主要功能
1. **动画控制**:
   - 页面加载动画
   - 滚动触发动画
   - 悬停效果控制
2. **交互功能**:
   - 功能演示器
   - 模型选择器
   - 语言切换器
3. **视觉效果**:
   - 背景粒子系统
   - 渐变动画
   - 数字滚动效果
4. **用户界面**:
   - 导航菜单控制
   - 模态框管理
   - 响应式布局调整

## 技术栈
- **HTML5**: 语义化标签，SEO友好
- **CSS3**: Flexbox/Grid布局，动画效果
- **JavaScript ES6+**: 现代语法，模块化开发
- **Tailwind CSS**: 快速样式开发
- **动画库**: Anime.js, p5.js, ECharts.js, Splide.js
- **字体**: Inter字体家族
- **图标**: Heroicons SVG图标
